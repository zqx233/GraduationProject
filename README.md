# GraduationProject
人脸识别系统设计与实现毕业设计代码

# 开发环境
* Python 3.9

# 主要功能
* 人脸识别与属性分析
* 人脸对比
* 人脸搜索
* 人脸库管理

# 设计实现

界面实现使用Tkinter库，人脸识别功能使用百度AI实现。

# 如何使用

首先需要在[百度AI](https://ai.baidu.com/)官网注册并创建应用，获取到API Key和Secret Key，填写到getaccesstoken.py中，运行后将返回结果中的access_token复制到function.py中，在安装好requirements.txt中的库后，即可运行main.py。

项目中还有一些未完善和可优化的地方。
